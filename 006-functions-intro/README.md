# Intro to Functions

In JavaScript functions can be created in 3 ways:

1. Function declaration
2. Function expression
3. Arrow function

<br>

### Function Declaration

```javascript
function name(parameters) {
  // function body
}
```

### Function Expression

```javascript
let name = function (parameters) {
  // function body
};
```

### Arrow function

```javascript
let name = (parameters) => {
  // function body
};
```
